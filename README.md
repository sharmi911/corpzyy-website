 Corpzyy Website

A modern frontend demo website created for interview demonstration purposes at Valaidhalam Technologies.

This project showcases a clean corporate-style UI design, responsive layouts, and modern web development practices.  
The website is developed as a frontend prototype/demo and does not include backend functionality.

## Purpose

The main purpose of this project is to:

- Demonstrate frontend development skills
- Showcase UI/UX design concepts
- Present a professional company-style landing page
- Use as a demo project during interview selection process at Valaidhalam Technologies

## Features

- Responsive Design
- Modern UI Layout
- Interactive Sections
- Clean Navigation
- Mobile-Friendly Interface
- Professional Corporate Theme

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Status

This project is a frontend demo/prototype created only for interview and presentation purposes.

Backend services, database integration, and production-level functionalities are not included.

## Installation

Clone the repository:

```bash
git clone https://github.com/sharmi911/corpzyy-website.git
````

Go to the project folder:

```bash
cd corpzyy-website
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
